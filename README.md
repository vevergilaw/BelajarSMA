# 📚 BelajarSMA - Platform Edukasi SMA & TKA

**Belajar Lebih Cerdas, Nilai Lebih Tinggi**

Platform edukasi interaktif untuk siswa SMA dan persiapan TKA dengan materi lengkap, latihan soal, dan simulasi ujian.

## ✨ Fitur Utama

- 📖 **Materi Lengkap** - 10 mata pelajaran SMA dengan ratusan topik
- 🎯 **Latihan Soal Interaktif** - Soal pilihan ganda dengan pembahasan instan
- 📊 **Simulasi TKA** - 25 ronde ujian × 20 soal dengan timer countdown
- 👤 **Dual Role System** - Dashboard terpisah untuk Siswa & Guru
- 📈 **Progress Tracking** - Monitor perkembangan belajar real-time
- 🎨 **Design Modern** - UI/UX bersih, mobile-friendly, profesional

## 🛠️ Tech Stack

- **Frontend**: React 18 + TypeScript + Tailwind CSS
- **Backend**: Node.js + Express + PostgreSQL
- **Authentication**: JWT
- **State Management**: Redux Toolkit
- **Database**: PostgreSQL

## 🚀 Instalasi

### Frontend
```bash

npm install
npm start
```

### Backend
```bash
cd backend
npm install
npm run dev
```
